### `INSTALACION TERMUX`
```bash
> termux-setup-storage
> apt update && apt upgrade -y -y
> apt install git -y
> apt install nodejs -y
> apt install ffmpeg -y
> apt install imagemagick -y
> git clone https://github.com/SinNombre999/Aux-Bot
> cd ZenBot
> npm install
> npm install -g npm@8.5.3
> npm update
> npm start
```
